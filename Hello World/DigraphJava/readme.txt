This Java application was created in IntelliJ IDEA 15.0.2
This Java application is a console application.

//////////////////Getting Started///////////////////////////
The soucre code to this project has also been exported to:
DigraphJava\exportToHTML\com\Train\route in HTML format. 

To run this application in another IDE copy input.txt located 
in the root dir of your project.

Then copy the java files to your project, the path should
look something like the one below:
DigraphJava\src\com\Train\route

Run applicaton from IDE.

If you would like to change route distances adjust
values in input.txt file in project root directory.
